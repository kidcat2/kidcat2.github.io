#### queue 와 stack 비우기

```cpp
queue <int> q;
stack <int> s;
// q.push() ..  s.push()..
q = {}, s ={} ; // = {} 를 통해 queue와 stack을 empty로 만들 수 있다.
```


#### i != n 일 때는 " " 를 i == n 일 때는 "\n"을 출력하기

```cpp
for (int i = 0; i <= n; i++) {
    //code..
    cout << "1" << " \n"[i == n];
}
```
i= 0 ~ n-1 일 때는 1 1 1 1 ... 로 출력하고, i = n 이 되면 '\n'(줄바꿈)이 된다.
